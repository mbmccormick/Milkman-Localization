### Application Title

Milkman


### Featured Application Description

Remember The Milk per WP7


### Detailed Description

Milkman è il modo più veloce e più pulito per gestire le attività sul vostro Windows Phone. Milkman si sincronizza con Remember The Milk, un sistema di gestione di attività e di scadenze online. È possibile visualizzare le attività, creare notifiche ed aggiornare le proprie liste con tutta facilità, direttamente dal telefono. Ogni singola modifica fatta con Milkman viene sincronizzata con l'account Remember The Milk, in modo da non farvi mai più dimenticare il latte (e tutto il resto).

Milkman si integra perfettamente con il telefono, visualizzando i contatori delle attività su una live tile. Vengono mostrate le relative notifiche quando ci sono attività in scadenza ed è anche possibile utilizzare le notifiche basate sulla propria posizione geografica.

Questa applicazione utilizza le API di Remember The Milk, ma non è approvata né certificata.
